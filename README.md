# Terraform Workshop

Terraform과 Helm을 활용한 AWS EKS 클러스터 환경을 구축합니다.

### 주요 기능

- VPC부터 EKS 클러스터까지 전체 인프라 구축
- Helm Chart를 통한 애플리케이션 배포 자동화
- IAM 권한 및 보안 그룹 설정
- **IaC**: Terraform
- **패키지 관리**: Helm
- **환경**: AWS EKS
